# AI Resume–Job Matching System

This is a runnable starter scaffold for a full‑stack AI resume → job matching system.

Services:
- backend: Node.js + Express (TypeScript)
- ml-service: Python FastAPI with sentence-transformers
- frontend: React + Vite
- docker-compose.yml to run everything locally (Postgres + Redis included)

See docker-compose.yml and migration SQL to initialize DB (enable pgvector).
